Pygame Tetris (Enhanced Version)
🕹️ 俄罗斯方块增强版
📌 Overview | 项目介绍
This is a desktop mini-game built with Python and the Pygame library. It features a complete game loop, UI systems, and customized logic.

本项目是基于 Pygame 开发的俄罗斯方块增强版，实现了完整的游戏循环、UI 系统及优化的逻辑判定。

🌱 My Programming Enlightenment: This project marks the very beginning of my coding journey and my passion for software development.

我的编程启蒙： 这个项目标志着我编程之路的开端，也是我开发热情的起点。

✨ Key Enhancements | 核心改进
Derived from the core logic of , I have implemented the following enhancements:
在原版逻辑基础上，本项目增加了以下特性：

Advanced Collision Detection: Refined movement and rotation constraints.

Game Mechanics: Multi-row elimination and boundary constraints.

UI/UX: Added Start/Game Over screens, pausing, and navigation menus.

Audio: Integrated background music for different game states.

📂 Project Structure | 文件结构
main.py: Entry point; manages window lifecycle.

game.py: Core game loop and logic.

block.py & blockGroup.py: OOP implementation for tetrominoes.

menu.py: UI menu system.

const.py & utils.py: Constants and helper functions.

🛠️ How to Run | 运行说明

From Source: Run python main.py (Pygame required).
